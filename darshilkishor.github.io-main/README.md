# darshilkishor.github.io

A Pirate-themed Developer Portfolio featuring a custom admin dashboard and a MongoDB backend for reading and writing contact messages!

## Core Features
- **Frontend**: Custom stylized HTML & CSS with pirate thematic elements and custom cursor.
- **Backend**: Node.js and Express.js with a MongoDB integration to store messages securely.
- **Admin Dashboard**: Hosted securely on a custom route.

## Getting Started
Please see `HOW_TO_RUN.md` for steps on installing dependencies and running the backend server locally. The environment requires Node.js and a running MongoDB instance.
